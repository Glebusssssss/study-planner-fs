# study-planner-fs
FocusFlow: Full-Stack Study Planner
FocusFlow is a minimalist, high-performance study management tool built to solve the problem of academic disorganization. Unlike simple to-do lists, this application utilizes a dedicated MySQL backend to ensure all tasks are stored persistently and securely.

Tech Stack
Frontend: HTML5, CSS3 (Modern Dark Mode UI),  JavaScript (ES6+).

Backend: Node.js, Express.js.

Database: MySQL.

API: RESTful architecture for handling data flow.

Key Features
Persistent Storage: Tasks remain saved even after browser refresh or server restart.

CRUD Operations: Full capability to Create, Read, Update (Toggle Completion), and Delete tasks.

Asynchronous UI: Used the Fetch API to communicate with the server without reloading the page.

Responsive Design: Optimized for both desktop and mobile study sessions.

Installation
Clone the repository.

Run npm install to install dependencies.

Configure your MySQL database using the provided database.sql schema.

Run node server.js to start the local environment at localhost:3000.
